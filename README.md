# rig‑22

A lightweight template toolkit for local testing and CI. It bundles reusable YAML snippets, linter/formatter setups, and small automation helpers that I can drop into any project.

## Features
- Framework‑agnostic templates for the stacks I work with
- Plug‑and‑play YAML configs for CI, linting, formatting, and local checks
- Designed for automation and repeatability
- Continuously refined as my workflows evolve

## Purpose
rig‑22 keeps my projects consistent, reproducible, and easy to spin up—whether I’m testing locally or wiring up a new CI pipeline.

## Status
Actively developed and frequently tweaked as I refine my engineering workflow.
